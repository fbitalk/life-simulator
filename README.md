# 人生模拟器 (Life Simulator)

一个基于浏览器的文字人生模拟游戏，让你体验不同的人生选择、收集成就标签，书写属于你的独特人生故事。

## ✨ 核心特色

*   **无限可能**: 每次人生都由随机的出身和事件构成，独一无二。
*   **标签驱动**: 你的所有特征都通过 **标签 (Tags)** 展现。
    *   **⚪ 白色**: 普通标签。
    *   **🔴 红色**: 负面状态（如：体弱多病、贫困）。
    *   **⚫ 黑色**: **轮回天赋**。死后可继承到下一世的稀有标签！
    *   **🟣 紫色**: 史诗/异能标签。
    *   **🌸 粉色**: 情感/恋爱标签。
    *   **🟡 金色**: 稀有成就标签。
*   **属性系统**: 健康、金钱、智力等属性动态变化，达到阈值会自动获得对应标签（"富有", "天才" 等）。
*   **人生回顾 (墓地)**: 死亡后，你的生平会被记录在墓地中，随时可以回顾。
*   **成就传承**: 解锁的成就和黑色天赋标签会自动保存。

## 🚀 如何运行

1. 下载或克隆本项目。
2. 直接在浏览器中打开 `index.html`。
3. 点击 **"新的人生"** 开始游戏。

> 纯前端项目，无需后台服务器即可运行。

## 📖 游戏规则

### 1. 开启人生 (0岁)
*   **初始状态**: 所有属性默认为 50。
*   **性别**: 随机获得【男性】或【女性】标签。
*   **天赋**: 继承上一世获得的【黑色标签】。
*   **出身**: 0岁时的第一个事件决定了你的家庭背景（富裕、贫穷、书香门第等）。

### 2. 成长与选择
*   **每年**: 点击"下一年"，随机触发一个事件。
*   **触发机制**: 为了保证逻辑通顺，事件是根据你的 **标签** 和 **优先级** 筛选的。系统会优先推送更重要的事件。
*   **抉择**: 不同的选项会改变你的属性（金钱、健康等），甚至改变你的人生轨迹（获得新标签）。

### 3. 生老病死
*   **自然老死**: 只有当你 **年满80岁** 且 **健康 < 30** 时，才会在年末判定是否老死。
*   **意外死亡**:
    *   **健康归零**: 任何时候健康 <= 0 会立即死亡（如：突发疾病、打架斗殴）。
    *   **风险事件**: 部分高回报选项带有 `risk` (风险)，运气不好会直接导致死亡。

### 4. 游戏结束
*   死亡后，你可以选择：
    *   **下葬**: 保存当前人生记录到墓地。
    *   **新的人生**: 带着继承的黑色天赋立即转世。
    *   **主菜单**: 返回首页。

## 📂 项目结构

```
life-simulator/
├── index.html          # 游戏主入口
├── event-system-guide.md # 开发者指南：如何编写新事件
├── css/
│   └── style.css       # 样式文件
└── js/
    ├── game.js         # 游戏核心控制器

    ├── modules/        # 功能模块 (常量、数据管理)
    └── events/         # 事件数据库
        ├── maleEvents.js    # 男性专属 & 开局事件
        ├── femaleEvents.js  # 女性专属 & 开局事件
        ├── ageEvents.js     # 各年龄段通用事件
        ├── commonEvents.js  # 普通标签事件
        ├── redEvents.js     # 红色负面事件
        └── ...              # 其他颜色事件
```
